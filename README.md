# BarCodeScanner
Java 接收扫码枪数据

* 首先要告诉你个秘密，扫码枪就是=键盘+回车，所以你建个txt文本文档或者doc随便，打开，然后连接扫码枪，开始扫描，扫码结果就会自动填入你的文档；
* 然后，这个项目要给你讲的是，java程序上接收扫码枪数据。
